📖 SOP Quick Reference

Standard procedures for handling each CRETS error type. Reference these during associate engagements.

Customer Info HIGH SEVERITY

  • All customer PII (name, address, phone, email) MUST be covered or removed before processing
  • Check all labels, packing slips, and inserts for visible customer data
  • Use black marker or label to cover — do NOT leave partially visible
  • If digital screen shows customer info, do not photograph or share
  • Escalation: Report to PA/AM immediately if customer data was sent downstream uncovered

Missing Item HIGH SEVERITY

  • Verify LPN contents against expected manifest
  • Check surrounding area and totes for misplaced items
  • If item cannot be located, escalate to Problem Solve immediately
  • Document the missing item with LPN and expected contents
  • Do NOT mark as complete if item is unaccounted for

Wrong Item MEDIUM SEVERITY

  • Compare item received against LPN label and system record
  • If item does not match, set aside and escalate to Problem Solve
  • Do NOT re-bin or re-slot a wrong item — it creates downstream errors
  • Document the discrepancy: what was expected vs. what was received

Broken Item MEDIUM SEVERITY

  • Inspect all items for visible damage during processing
  • Broken/damaged items MUST be escalated to Problem Solve — do NOT pass downstream
  • Grade appropriately: damaged items are NOT sellable
  • If item is leaking, see Hazmat Item procedures
  • Place damaged items in designated damage area with LPN attached

Hazmat Item HIGH SEVERITY

  • Any leaking, spilling, or fuming item MUST be escalated to Hazmat team IMMEDIATELY
  • Do NOT continue processing — isolate the item
  • Do NOT send downstream to other buildings
  • Alert nearby associates and PA/AM
  • Follow site-specific hazmat spill procedures
  • Failure to escalate creates safety risk and can result in unsellable inventory at receiving buildings

Multiple LPNs MEDIUM SEVERITY

  • Each container should have ONE LPN — multiple LPNs cause tracking errors
  • If multiple LPNs found, identify which is correct and remove extras
  • Escalate to Problem Solve if unable to determine correct LPN
  • Do NOT process with multiple LPNs attached — this causes inventory discrepancies

Sellable Polybag No Seal LOW SEVERITY

  • Sellable items in polybags MUST be properly sealed before being sent downstream
  • Check seal integrity — reseal if open or damaged
  • If no sealing equipment available, escalate to PA for supplies
  • Unsealed polybags can result in item damage, contamination, or customer complaints
